LxPhisher is a phishing-awareness simulation toolkit for controlled training and defensive security education.

## Setup and Running

Use this project only in isolated environments where all participants have given consent and where you have explicit authorization from the system owner.

For environment-specific setup details, use the project scripts and dependency files in this repository:

- `install_lxphisher.sh`
- `lxphisher.sh`
- `requirements.txt`
