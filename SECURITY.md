# Security Policy

## Supported Versions

Only the latest version of LxPhisher receives security updates.

## Reporting a Vulnerability

Do not post unpatched vulnerabilities publicly.

To report a vulnerability, email:

- Email: `voltsparx@gmail.com`
- Subject: `LxPhisher Security Vulnerability`
- Include: impact, reproduction steps, affected file(s), and proposed mitigation if available

Please allow up to 48 hours for an initial response.

## Responsible Disclosure

We follow coordinated disclosure. Please provide reasonable time for triage and remediation before public disclosure.

## Safe Testing Requirements

- Use only in isolated, authorized lab environments.
- Do not test against production systems without written approval.
- Minimize and securely delete any generated lab data after the exercise.
- Avoid changes that increase abuse potential or bypass consent controls.
